Моделирование предметной области

  1. Что из перечисленного не является одним из четырех типов ассоциации в доменной модели?

    1. Аггрегация (has-a)

    2. Создает

    3. Генерализация (is-a)

    4. Знает о

  2. Какой термин используется в случаях, когда дочерний класс расширяет класс-родитель?

    1. Аггрегация

    2. Наследование

    3. Композиция

    4. Инкапсуляция

  3. Нарисуйте предметную область онлайн-магазина книг

  4. Какие требования к системе описывают прецеденты?

    1. Функциональные требования

    2. Не функциональные требования

  5. На какие вопросы помогают ответить прецеденты?

    1. Что пользователи пытаются сделать с помощью системы?

    2. Как взаимосвязаны сущностей системы?

    3. Чем система может быть полезна пользователям?

    4. Требования к аппаратным ресурсам?

  6. Для чего служат альтернативные сценарии?

    1. Альтернативный сценарий включает в себя действия, которые пользователь хочет сделать что-то не так, как это предусмотрено в системе

    2. Альтернативный сценарий включает в себя действия, которые пользователь вынужден совершать, когда что-то пошло не так.

  7. Выберите правильный способ написания прецедента. Почему данный способ правильный?

    1. Пользователям должна быть обеспечена возможность авторизоваться на сайте с помощью логина и пароля.

    2. Пользователь вводит логин и пароль и затем нажимает кнопку Войти. Система ищет данные пользователя по логину и проверяет совпадение пароля. Затем система пускает пользователя.

  8. Выберите и объясните правила при составлении прецедентов

    1. Нужно писать текст прецедента от первого лица

    2. Нужно писать текст прецедента от третьего лица

    3. Текст прецедента должен быть построен по принципу: событие/отклик системы на него

    4. Текст прецедента не должен включать отклики системы на происходящие события

    5. Классы предметной области должны быть явно упомянуты в тексте прецедентов

    6. Прецеденты могут оперировать произвольными терминами и их синонимами
